About Me

I'm Rahul Sharma

Full Stack & Mobile App Developer

I’m a Full-Stack Developer experienced in building scalable, production-ready web and mobile applications using React, Next.js, and React Native. My work focuses on creating clean, maintainable frontend architectures, intuitive user flows, and responsive interfaces that align with real business and product requirements.

I’ve worked extensively on B2B and B2C platforms in the travel and mobility domain, contributing to features such as booking workflows, payment integrations, search and discovery experiences, and vendor-facing systems. My core skill set includes JavaScript, TypeScript, Redux, Zustand, React Hook Form, Zod, and Tailwind CSS.

I also have a solid working understanding of backend technologies including Node.js, Express, MongoDB, and MySQL, enabling effective API integration and collaboration across teams. I enjoy taking ownership of features, improving UI/UX through thoughtful iteration, and delivering reliable, user-centric products.
